# netstat.current.tcp.connection.node.sh

nodejs active tcp connection lookup command line utility

## usage

```bash

npm i -g @m1yh3m/netstat.current.tcp.connection.node.sh
active_tcp <sortBy>

# where sortBy is one of protocol|host|port

```
